
## Gem D3 : Bubble Chart

Create a bubble chart of Bay Area cities. Longitude and latitude are XY.

The size of the bubble is based on population (2020) and a tooltip displays the city name.

<img src="2602-gd3-bubble_chart-01.jpg" width=400px>

The prompt :

```
Write a Javascript function to generate a d3.js bubble chart for the input data file bay_cities-lon_lat_pop.tsv.
The x-value should be the longitude column.
The y-value should be the latitude column. Used fixed aspect ratio for the coordinates.
The size of the bubble should be based on the Population (2020) column.
Include a tooltip to display the value of the Name column when the mouse is over the corresponding bubble.
The function should take input parameters of input data file name, plot width and plot height.
Generate the JavaScript code and save to file bubble_chart.js.
Also generate an index.html file to call this javascript function. 
Do not start any processes to install or invoke an http server.
```
