
## Gem D3 : Force-Based Graph

Create force-based graph of the network of Alice In Wonderland characters.

Edges are links are for mentions of the character in the same chapter.

<img src="2603-gd3-force_based_graph-01.jpg" width=400px>

The prompt:

```
Write a Javascript function to generate a d3.js force-directed network graph for the input data file bay_alice.json.
The function should take input parameters of input data file name, plot width and plot height.
This data consists of nodes that are characters in Alice In Wonderland.
The edges or links in this data are chapters in which both characters are mentioned.
The graph nodes should be labeled.
Generate the JavaScript code and save to file: force_directed_graph.js.
Also generate an index.html file to call this javascript function.
Do not start any processes to install or invoke an http server.
```
