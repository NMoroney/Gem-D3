
## Gem D3 : Donut Plot

Create a labeled donut plot of the bay cities population data.

<img src="2603-gd3-donut-01.jpg">

The prompt :

```
Write a Javascript function to generate a d3.js labeled donut chart for the input data file bay_cities-top_20_pop.tsv.
The function should take input parameters of input data file name, plot width and plot height.
Generate the JavaScript code and save to file: donut_plot.js.
Also generate an index.html file to call this javascript function. 
Do not start any processes to install or invoke an http server.
```
