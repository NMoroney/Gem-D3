
## Gem D3 : Ordered Horizontal Lollipop 

For the top 20 bay area cities by population, create an ordered horizontal lollipop (dot) plot.

<img src="2603-gd3-ordered_horizontal_lollipop-01.jpg" width=600px>

The prompt :

```
Write a Javascript function to generate a d3.js ordered horizontal lollipop chart for the input data file bay_cities-top_20_pop.tsv.
The function should take input parameters of input data file name, plot width and plot height.
Generate the JavaScript code and save to file. ordered_horizontal_lollipop.js.
Also generate an index.html file to call this javascript function. 
Do not start any processes to install or invoke an http server.
```
