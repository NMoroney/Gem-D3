
## Gem D3 : Heat Map

Generate a heat map of Alice in Wonderland character mentions.

<img src="2603-gd3-heat_map-alice-01.jpg">

The prompt :

```
Write a Javascript function to generate a d3.js heat map visualization for the input data file alice.csv.
The first column or chapter is the group or x-axis of the heat map.
The second column or character is the variable or y-axis of the heat map.
The third column or count is the value be plotted.
The function should take input parameters of input data file name, plot width and plot height.
Generate the JavaScript code and save to file. heat_map.js.
Also generate an index.html file to call this javascript function. 
Do not start any processes to install or invoke an http server.
```
